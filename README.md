## E-Commerce Sales Analysis Dashboard (Power BI)

This project focuses on analyzing e-commerce sales data and presenting meaningful insights through an interactive Power BI dashboard. The solution is designed to support data-driven decision-making by enabling users to explore sales performance across multiple dimensions in a simple and intuitive way.

### Project Overview

- Designed and developed an interactive dashboard to monitor, track, and analyze online sales performance.
- Implemented advanced parameters to enable dynamic drill-down across reports, enhanced with custom filters and slicers for improved user interaction.
- Built and managed data connections, performed table joins, and created calculated fields to transform and manipulate raw data into actionable insights.
- Enabled user-driven parameters to allow flexible customization of visuals and deeper analytical exploration.
- Delivered insights using a wide range of customized visualizations, including:
  - Bar Charts
  - Pie and Donut Charts
  - Clustered Bar Charts
  - Scatter Plots
  - Line and Area Charts
  - Maps
  - Interactive Slicers

### Tools & Technologies

- Power BI
- DAX
- Data Modeling
- Data Transformation
- Interactive Data Visualization

### Key Outcome

The dashboard provides a comprehensive view of e-commerce sales performance, helping stakeholders quickly identify trends, patterns, and key business insights through interactive and visually rich reports.
